```
📘 파이썬 Pandas & Numpy 수업 요약 – 초간단 정리 🐍

🧩 [1교시] Pandas 기본
───────────────────────
✅ 설치 과정 (uv 사용)
1. uv init  
2. uv가 설치된 폴더로 이동 (cd)  
3. uv add pandas  
4. import pandas as pd → 버전 확인: `print(pd.__version__)`

✅ Series 객체 생성
- 리스트를 기반으로 생성: `pd.Series(data=[1,2,3])`
- 인덱스 지정 가능: `pd.Series(data=[1,2,3], index=["a","b","c"])`

✅ 데이터 접근
- `iloc[]`: 정수 인덱스로 접근  
- `loc[]`: 인덱스 라벨로 접근  
예) `data.loc["3"]`, `data.iloc[1]`

✅ 슬라이싱
- `data[:2]` → 인덱스 0,1까지 접근  
- 리스트처럼 슬라이싱 가능

✅ 일반 리스트와 비교
- `b = [1,2,3,4]` 와 같은 방식으로 접근 가능  
- 단, 리스트는 `b[[1,2]]` 불가, Series는 가능

🧩 [2교시] 문자열, 숫자 데이터, DataFrame 사용
───────────────────────
✅ 다양한 데이터 타입 가능
- 문자열도 Series에 사용 가능  
  예: `pd.Series(["가","나","다"])`

✅ Series에 직접 리스트 삽입
- `pd.Series([1.0, 2.0, 3.0], index=["가","나","다"])`

✅ DataFrame 생성
arr = [
  [1,2,3],
  [4,5,6],
  [7,8,9]
]
data = pd.DataFrame(arr)

✅ DataFrame에 행/열 이름 부여
pd.DataFrame(data=arr, index=["1행","2행","3행"], columns=["1열","2열","3열"])

✅ DataFrame에서 특정 행 추출 시 → Series로 반환됨
- 2차원 중 일부를 꺼내면 자동으로 1차원으로 줄어듦

🧩 [3교시] Numpy, CSV 읽기, 연산
───────────────────────
✅ Numpy 기본
import numpy as np
data = np.array([1,2,3,4,5])
print(data[data > 3])  # 조건 필터링 가능
print(data * 3)        # 요소별 연산 가능

✅ CSV 파일 읽기 (pandas)
data = pd.read_csv("./data1.csv", index_col=0)

- `index_col=0`: 첫 번째 열을 인덱스로 사용하겠다는 뜻

✅ 데이터 접근 및 반복
for i in data.index:
    s = data.loc[i]
    dan = s.loc["dan"]
    num = s.loc["num"]
    print(f'{dan} * {num} = {dan * num}')

- `data.index` → 인덱스 목록  
- `data.loc[]` → 행 접근  
- `s.loc["dan"]` → 열 값 접근

───────────────────────────────────────────────
📌 추가 개념 요약
───────────────────────────────────────────────
🔹 Series: 1차원 데이터 (리스트 + 인덱스)  
🔹 DataFrame: 2차원 데이터 (표 형태)  
🔹 loc[]: 라벨 기반 접근  
🔹 iloc[]: 숫자 인덱스 기반 접근  
🔹 슬라이싱, 리스트 인덱싱 모두 지원  
🔹 넘파이: 빠른 수치 연산, 조건 필터링, 브로드캐스팅 가능  
🔹 csv 파일: `read_csv()`로 불러오기, 인덱스 지정 필수 아님
```

```
📘 Pandas & Numpy 핵심 개념 요약 🧠  
───────────────────────────────────────────────

🔹 1. Series vs DataFrame 차이
───────────────────────────────
| 항목   | Series                  | DataFrame                     |
|------- |------------------------ |------------------------------ |
| 구조   | 1차원                    | 2차원 (행과 열)                |
| 구성   | 값 + 인덱스              | 열 이름 + 행 인덱스 + 값        |
| 생성   | pd.Series([1, 2, 3])     | pd.DataFrame([[1, 2], [3, 4]])|

👉 Series는 열 하나짜리 DataFrame처럼 보이지만 다름!  
   연산 시 `type()`으로 확인하는 습관 들이기.

🔹 2. loc[] vs iloc[] 차이
───────────────────────────────
| 구분    | loc[] (라벨 기반)     | iloc[] (숫자 인덱스 기반)  |
|---------|------------------------|-----------------------------|
| 예시    | data.loc["1행"]        | data.iloc[0]               |

👉 `loc`는 인덱스 이름, `iloc`는 숫자 위치로 접근  
❗ 혼동해서 틀리는 경우 많으니 꼭 구분해서 사용!

🔹 3. 슬라이싱 규칙
───────────────────────────────
data[1:3] → 인덱스 1, 2까지 출력 (3은 제외)

👉 끝 인덱스는 포함 ❌ (파이썬과 동일 규칙)

🔹 4. CSV 읽을 때 index_col=0 의미
───────────────────────────────
pd.read_csv("파일.csv", index_col=0)

✅ 첫 번째 열을 행 인덱스로 사용하겠다는 의미  
    (예: ID, 이름 등 고유한 열일 때 유용)

🔹 5. Numpy는 요소별 연산 가능
───────────────────────────────
np.array([1, 2, 3]) * 3  
→ 결과: [3, 6, 9]

👉 리스트는 곱하면 반복되지만, Numpy는 벡터화 연산이 일어남  
    속도도 빠르고 연산도 간편!

🔹 6. DataFrame 반복 기본 패턴
───────────────────────────────
for i in data.index:  
    row = data.loc[i]  
    dan = row.loc["dan"]  
    num = row.loc["num"]  
    print(f'{dan} * {num} = {dan * num}')

✅ `.index`로 반복 → `.loc[]`로 행 접근 → `.loc["열이름"]`으로 열 접근  
    → DataFrame을 다룰 때 매우 자주 사용되는 핵심 패턴

🔹 7. 타입 확인은 디버깅의 시작
───────────────────────────────
type(data)             → DataFrame  
type(data.loc["1행"])  → Series
```
